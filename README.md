# Keter Marketing

Site vitrine (Vite + React + TypeScript) avec motion (Framer Motion) et smooth scroll.

## Développement local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Déploiement Vercel

- Build command: `npm run build`
- Output directory: `dist`
- Sitemap: `/sitemap.xml`
- Robots: `/robots.txt`
